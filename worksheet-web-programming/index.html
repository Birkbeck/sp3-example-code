Exercise 1: Install client-side REST tools


In this exercise, you are going to download and install REST client tools.  You can install either Postman or RestClient.
  1. Download and install "curl"
  2. Install Postman Chrome plugin (or desktop version of it)
  3. Install "RestClient"

(1.1) Download "curl" (only for systems which do not have it already)


For Mac OS misc.X or other Linux OS'es, "Curl" utility comes with the OS.  Re-installation of "Curl" might cause some malfunctioning behavior on Mac OS misc.X or other Linux OS'es.

0. Check if curl is already installed on your system
  • In a Terminal or console, type curl 
1. Using your browser, go to http://curl.haxx.se/.


  • Download the latest stable binary and library corresponding the your system
  • Install them on your machine

(1.3) Download and install "RestClient" (Optional)


RestClient is a Java-based REST client tool.

1. Using your browser, go to http://jpassion.com/downloads/restclient-ui-3.2.2-jar-with-dependencies.jar  and download it.

2. Run it using the following command. (Or you can double click the restclient-ui-3.2-jar-with-dependencies.jar within the file system explorer of your OS.)

java -jar restclient-ui-3.2-jar-with-dependencies.jar



3. You can do set "Auto-indent Response Body" option to see XML/JSON responses in properly indented format.  (Otherwise, the XML/JSON response will be a single line format, which would be hard to read.)
